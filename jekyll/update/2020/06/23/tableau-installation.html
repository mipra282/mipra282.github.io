Guide to Install MongoDB on Ubuntu 16.04 Server and Connect with Tableau Desktop

MongoDB Community and MongoDB Mysqlid Install

https://docs.mongodb.com/manual/tutorial/install-mongodb-on-ubuntu/

Steps (On ubuntu/linux server):

  1. $wget -qO - https://www.mongodb.org/static/pgp/server-4.2.asc | sudo apt-key add -
  2. $sudo apt-get install gnupg
  3. $wget -qO - https://www.mongodb.org/static/pgp/server-4.2.asc | sudo apt-key add -
  4. $echo "deb [arch=amd64,arm64] https://repo.mongodb.org/apt/ubuntu xenial/mongodb-org/4.2 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.2.list
  5. $sudo apt-get update
  6. $sudo apt-get install -y mongodb-org
  7. $echo "mongodb-org hold" | sudo dpkg –set-selections
  8. $echo "mongodb-org-server hold" | sudo dpkg –set-selections
  9. $echo "mongodb-org-shell hold" | sudo dpkg –set-selections
  10. $echo "mongodb-org-mongos hold" | sudo dpkg –set-selections
  11. $echo "mongodb-org-tools hold" | sudo dpkg –set-selections
  12. $sudo systemctl daemon-reload
  13. $sudo systemctl start mongod
  14. $sudo systemctl status mongod
  15. $sudo systemctl enable mongod #Adds as a service to system boot
  16. $sudo systemctl stop mongod
  17. $Access mongo shell using command:

$mongo

Go to /etc/mongod.conffolder and change bindIp to 0.0.0.0.Save and exit.

$vi /etc/mongod.conf

Next install the BI connector on Linux server which will install mongsqlid which provides a proxy to tunnel your Tableau connections to mongodb database on the server.

https://docs.mongodb.com/bi-connector/master/tutorial/install-bi-connector-debian/#install-the-bi-short

$tar -xvzf mongodb-bi-linux-{arch}-{platform}-{version}.tgz

Eg. $wget https://info-mongodb-com.s3.amazonaws.com/mongodb-bi/v2/mongodb-bi-linux-x86_64-ubuntu1604-v2.13.4.tgz

Next, extract the archive and cd into bin folder

Next hit this command,

$./mongosqld –mongo-uri "mongodb://104.154.35.52:27017/?connect=direct"

Tableau Desktop Installation and Connect Mongodb server

Download Tableau latest version trial from this link:

https://www.tableau.com/trial/tableau-software?utm_campaign_id=2017049&utm_campaign=Prospecting-PROD-ALL-ALL-ALL-ALL&utm_medium=Paid+Search&utm_source=Bing&utm_language=EN&utm_country=IND&kw=%2Btableau%20%2Bdesktop&adgroup=CTX-Brand-Tableau+Desktop-EN-B&adused=&matchtype=p&placement=&&msclkid=da239555054e161b4237cbbe1d065f12&gclid=CIratJvsiuoCFUU8jgodxOYN7Q&gclsrc=ds

Register and start trial.

Now download the driver for Tableau MongoDB BI connector which internally uses Mysql 8 drivers.

https://www.tableau.com/support/drivers

Also go to mysql driver page and install Mysql 8 ODBC Connector for your OS.

Image source:

Next,

Initiating the connection using Tableau.

  1. Start Tableau Desktop
  2. Go to More option under Connect.

  1. Search for MongoDB BI Connector.
  2. Enter internal ip of your server and port as 3307 on the popup box

Hurray,you are connected now.
